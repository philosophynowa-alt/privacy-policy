Privacy Policy for Fit Era

Fit Era (package name: com.foxogaming.fitapp) is developed and published by BSAppk.

Fit Era respects your privacy. The app does not collect, store, or share personal user data on external servers.

The app requests the following permissions:
- Location: Used to track workouts and calculate activity data accurately.
- Notifications: Used to send workout reminders and motivational notifications.
- Media / Gallery Access: Used only to allow users to select or upload images within the app when needed.

All data accessed by these permissions is processed locally on the user's device and is not shared with any third parties.
